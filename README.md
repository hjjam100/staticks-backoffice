# Staticks-backoffice

스태틱스 백오피스 프로젝트

## 설명

[staticks-server](https://github.com/staticks/staticks-server) 를 백엔드로 하여, 유저가 페이지를 쉽게 구성할 수 있도록 하는 CMS 백오피스 프론트엔드
